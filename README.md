# OptiBot Website

This folder contains the complete, standalone presentation website for OptiBot.

## Hosting for Free (GitHub Pages)

This website is 100% static (HTML/CSS/JS) and can be hosted completely for free.

1. Create a new repository on [GitHub](https://github.com/).
2. Upload the contents of this `website/` folder to the repository.
3. In the repository settings, go to **Pages**.
4. Select `main` branch as the source.
5. Your website is now live!

Note: Ensure you include `OptiBot_Setup.exe` in the repository so the download button works.
